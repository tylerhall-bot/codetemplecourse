CSS Box Model Practice

Instructions:

Each box below has the same content size (200px × 100px) but different padding, borders, and margins. Notice how the total space each box occupies differs.

To inspect: Right-click any box → "Inspect Element" → Look for the "Box Model" diagram in developer tools.

Box 1
Content: 200×100px
Padding: 10px
Border: 2px solid
Margin: 20px
Box 2
Content: 200×100px
Padding: 25px
Border: 5px dashed
Margin: 10px
Box 3
Content: 200×100px
Padding: 5px
Border: 8px solid
Margin: 40px

What to observe:

  • Total width of Box 1: 200px (content) + 20px (padding) + 4px (border) = 224px
  • Total width of Box 2: 200px (content) + 50px (padding) + 10px (border) = 260px
  • Total width of Box 3: 200px (content) + 10px (padding) + 16px (border) = 226px
  • Margins create additional space around each box but don't affect the element's size
